### Prerequisites
- **Node.js** (v16+)
- **Supabase Account**
- **Cloudinary Account**
- **Git**
- **Python 3.10+ (for Flask AI Server)**

### Clone the Project
```
git clone https://github.com/StyNW7/Elysium.git
cd elysium
cd Client
npm install
npm run dev
```
